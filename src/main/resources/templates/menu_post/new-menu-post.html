새로운 메뉴 추가

메뉴 추가 버튼을 클릭하면 새로운 메뉴 항목을 손쉽게 추가할 수 있는 상자가 나타납니다. 아래의 입력란을 사용하여 메뉴 항목의 세부 정보를 정확하게 입력해주세요.

  • 이미지: 이미지를 삽입해주세요. [ 가능하면 이미지의 크기는 100 x 100으로 통일해주세요 ]
  • 가격: 메뉴 항목의 가격을 입력해주세요. [EX : 5000]
  • 카테고리: 메뉴 항목을 속하는 카테고리를 선택하거나 입력해주세요.[EX : 커피]

메뉴 항목을 추가하고 나면, 아래에 목록으로 정리되어 편리하게 확인할 수 있습니다. 추가하려는 메뉴 항목의 정보를 정확하게 입력하여 메뉴판을 업데이트해보세요.
